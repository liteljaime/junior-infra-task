# Metrics to monitor by service

### HTTP API

* Uptime
* CPU Usage
* Memory Usage
* Request per minute
* Average Max Latency
* Errors per minute

### MongoDB

* Replication lag
* Replica State
* Locking State
* Disk Utilization
* Database Connections

### Redis

* Throughput
* CPU usage
* Memory utilization
* Cache Hit Ratio
* Active Connections
* Replication metrics
* Expired/evicted key stats
* Persistence stats

### Postgres

* Disk and index usage details
* Read / Write query throughput
* Replication and reliability
* Resource utilization

### RabbitMQ

* CPU stats
* Memory usage
* Disk I/O
* TCP connections by state
* Network throughput
* Network latency
